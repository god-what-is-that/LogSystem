## 📋 二、具体规则

### 1. 🚫 禁止发送不适内容
- 🔞 严禁发送引起他人强烈不适的聊天记录、图片、视频等内容
- 🙅‍♂️ 包括但不限于血腥、暴力、恐怖等令人不适的内容
- 🎧 禁止发送含有炸麦视频、强光爆闪内容、跳杀内容的视频、动图或聊天记录

### 2. 🤝 反歧视与文明交流
- ⚖️ 严禁对某类群体或个人发表歧视言论
- 🚷 禁止人身攻击和引战行为
- 🔞 严禁多次发表擦边、低俗、违反公序良俗的内容

### 3. 🔒 信息安全与法律合规
- 🇨🇳 禁止政治敏感、违法、谣言信息
- 👤 禁止侵犯他人隐私
- 📢 严禁任何形式的广告，私下金钱交易后果自负
- 📹 禁止搬运其他up视频 (如要分享，请使用平台自带分享功能进行分享)
- 🔗 不得发送不明链接、二维码，如需发送请注明内容
### 4. 🏛️ 群内秩序管理
- 📢 禁止任何形式的刷屏、恶意灌水
- 🤝 纠纷应私聊或联系管理调解，禁止煽动对立
- 🗣️ 禁止骚扰他人
- 👥 禁止推荐其他群聊和拉人进其他群聊
- 🤖 禁止私自部署机器人或脚本
- ⚠  禁止讨论心理疾病、LGBTQ等敏感话题，若影响较大、劝阻无效或多次讨论，按违规处理。
- 🧨 禁止在群内传播不良思想引导，一经发现按严重违规处理。
- 😔 禁止在群内持续宣泄个人极端负面情绪

### 5. 🎨 同人创作守则
#### 📝 基本要求
- ❌ 禁止CP向二创和过激发言
- ✋ 不要因同人属性而互相攻击
- 🚫 禁止因他人的二创作品而攻击他人

#### 🖼️ 相册管理
| 相册类型   | 上传要求     | 注意事项       |
| ------ | -------- | ---------- |
| 个人相册   | 24小时内≥5张 | 标注【个人】，限1个 |
| 其他个人作品 | 与匿名兲无关   | 需为原创内容     |
| 兲のAI整活 | AI创作作品   | 注明AI生成     |
| 兲の表情包  | 表情包类     | 需清晰可用      |
| 兲の接头霸王 | P图作品     | 禁止侵权内容     |

### 6. 📞 举报机制
- 👮 发现违规行为可@管理员或群主举报
- 📸 需提供完整证据（截图/记录）

## ⚖️ 三、违规处理标准

### 📊 违规等级判定表

| 等级     | 违规类型 | 示例行为         | 处理措施     |
| ------ | ---- | ------------ | -------- |
| 🟢 轻微  | 一般违规 | 轻微刷屏、误发广告    | 警告+撤回    |
| 🟡 一般  | 重复违规 | 二次违规同类行为     | 禁言1-24小时 |
| 🟠 严重  | 特殊违规 | 引战、歧视言论      | 移出群聊+禁言  |
| 🔴 极严重 | 重大违规 | 违法、色情内容、不良思想 | 永久拉黑     |
### 🎯 具体执行标准

| 违规次数 | 处理措施 | 备注 |
|---------|---------|------|
| 第1次 | 📝 警告+撤回消息 | 情节严重可禁言1小时 |
| 第2次 | ⏰ 禁言24小时 | 同类违规行为 |
| 第3次 | 🚪 移出群聊 | 再次入群禁言1天 |
| 第4次 | 🚫 永久拉黑 | 不再接受入群申请 |
### ⚠️ 特殊违规即时处理

| 违规类型   | 处理方式    | 说明     |
| ------ | ------- | ------ |
| 政治敏感内容 | 🚫 永久拉黑 | 零容忍    |
| 色情淫秽内容 | 🚫 永久拉黑 | 包括性暗示  |
| 恶意人身攻击 | 🚫 永久拉黑 | 造成严重影响 |
| 非法机器人  | 🤖 立即移除 | 包括脚本程序 |
